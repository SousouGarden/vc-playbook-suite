**本文件包含 15 个 CEO 去留案例**：10 个 coup 发生的 + 5 个 CEO 成功守位的。每个按「Context / Board Structure / Coup Trigger / Coup Move / CEO Response / Outcome / Lessons」展开。

## 失败案例 1：Sam Altman × OpenAI（2023/11）

### Context
- OpenAI 成立为 nonprofit，后成立 for-profit subsidiary
- 2023 估值 $86B，融资多轮
- Sam 是灵魂人物但技术上 CEO 位置在 non-profit Board 下

### Board Structure
- Non-profit Board 6 人：Sam Altman, Greg Brockman, Ilya Sutskever, Adam D'Angelo, Tasha McCauley, Helen Toner
- Microsoft（$13B 投资）**没有 board seat**

### Coup Trigger
- Sam 的 commercial ambition vs safety focus 的张力
- Ilya 和 Helen Toner 对 Sam 的 "candor" 有 concern
- 具体 catalyst：Sam 推动 GPT store + custom GPTs 加速 commercialization

### Coup Move
- 11/17（周五）：Board 4 人（Ilya, Adam, Tasha, Helen）在 Sam 不在场时 vote 解雇
- 当天 fire Sam + 告诉 Greg Brockman 要 demote（Greg 辞职）
- Board 发 statement: "Sam was not consistently candid in his communications"
- 没有 investor 事先 notified（Microsoft 之前 15 分钟收到消息）

### CEO Response
- Sam 立即 pivot：join Microsoft + take employees with him
- 11/19-21：500+ OpenAI 员工联名要求 reinstate Sam（否则 follow him）
- Ilya 自己公开 regret："I deeply regret my participation in the board's actions"
- Microsoft CEO Satya Nadella 公开 signal: "whether Sam is back at OpenAI or joining Microsoft"

### Outcome
- 4 天后 Sam reinstated
- Board 重组：D'Angelo stay, 新 board 成员 Larry Summers + Bret Taylor + 后续加入
- Ilya / Tasha / Helen 出局
- Microsoft 获得 observer seat

### Lessons
- **"Employees will follow founder" is a nuclear option but it works**——Sam 的 response 本质是 "everyone leaves with me"
- **Board 没和 investors 同步 = governance failure**——Microsoft 后来成 power broker
- **Moral claims 需要 evidence**："not consistently candid" 被 Board 说出但没 public evidence → 变成 PR disaster
- **Single-round coup 无 process 会 backfire**——没有 progressive steps，直接 nuclear motion = 失败

### 对 founder 的教训
- 你的 **employee loyalty** 是 ultimate trump card
- Board 的 integrity failure 会被公众 punish
- **Communication 和 alignment 是最大 gift**——不要让 board 觉得他们 need 去 guess

## 失败案例 2：Travis Kalanick × Uber（2017）

### Context
- Uber 是 2017 年全球最高估值 private tech company（$70B）
- 一系列 scandal：sexual harassment、Greyball、God View、竞争对手 IP 纠纷
- Travis 被广泛视为 "culture problem"

### Board Structure
- Board 11 seats（比 typical 大很多）：Travis + Ryan Graves + Arianna Huffington + Saudi's PIF + multiple VCs (Benchmark, Kleiner, TPG, First Round, GV) + independent
- Benchmark (Bill Gurley) 是核心

### Coup Trigger
- 2017 年 2 月 Susan Fowler blog post（sexual harassment）
- 6 月 Eric Holder report 建议 organizational changes
- Travis 的母亲意外去世 → emotional context
- Accumulation of scandals → investor pressure

### Coup Move
- 6/20/2017：Benchmark 的 5 个 investors 发 letter demanding Travis resign
- Public pressure + board pressure coordinated
- Travis 6/21/2017 resign as CEO (but stay on board)

### CEO Response
- Initially try to fight - hire lawyers
- Then capitulate - sign resignation
- Retain board seat + voting rights
- Later sue Benchmark for breach of fiduciary duty

### Outcome
- Travis permanently out as CEO
- Dara Khosrowshahi become CEO
- Travis eventually sell majority of shares after IPO
- Benchmark vs Travis legal battles continue into 2019

### Lessons
- **Accumulation of scandals = death spiral**——single issue 可能 survive，多 scandal 不行
- **"Culture" becomes 武器**——when investors want you out, they frame it as culture
- **Board seats post-exit = partial compensation**——Travis 保住 voting rights 和 board seat 重要
- **Public pressure amplifies investor leverage**——Susan Fowler 一篇 blog post 改变 game

### 对 founder 的教训
- **Scandals 会 compound**——1 个 issue 可以 manage，3 个 issue 变成 existential
- **Investor letter is a nuclear option that works**——once 发出来，resistance 几乎 futile
- **Negotiate exit terms before capitulation**——hire lawyer 尽快

## 失败案例 3：Steve Jobs × Apple（1985）

### Context
- Apple Computer 成立 1976，Steve Jobs 和 Steve Wozniak
- 1985 Jobs 和 John Sculley（CEO）冲突
- Jobs 领导 Macintosh division，Sculley 领导 Apple II

### Board Structure
- Board 偏 professional management
- Jobs 有 founder equity 但不是 majority
- Key independent directors 支持 Sculley

### Coup Trigger
- Jobs 在 Macintosh division 管理混乱
- Sales 失望 + 产品 bugs
- Jobs 试图在 board meeting 发动政变 against Sculley
- Sculley 的 counter-attack

### Coup Move
- Jobs 在 board meeting 提议 remove Sculley + 自己接管 operations
- Sculley 反击，board 支持 Sculley
- Jobs 被 stripped of operational responsibilities
- Jobs resign

### CEO Response
- Initially fight - propose Sculley removal
- Lose vote
- Resign and start NeXT

### Outcome
- Jobs out 11 years
- NeXT + Pixar success
- 1996 Apple acquire NeXT
- 1997 Jobs return as CEO
- 2001+ Apple becomes most valuable company

### Lessons
- **Internal coup 需要 votes**——Jobs 没 count the votes
- **Founders without majority = replaceable**——即使 visionary
- **Exile can be productive**——NeXT + Pixar 让 Jobs 成熟
- **Return requires company's failure**——Apple 需要 Jobs 回来 save it

### 对 founder 的教训
- **Don't propose removal unless you have the votes**
- **Founder without majority control = professional management's captive**
- **Being fired can be the best thing**——use exile time wisely

## 失败案例 4：Parker Conrad × Zenefits（2016）

### Context
- Zenefits: 最快达 $1B valuation 的 SaaS
- 2015 estimated $4.5B
- Regulatory issues: 销售未取得保险 broker license 的员工

### Board Structure
- Andreessen Horowitz 主导
- Marc Andreessen + partner David Sacks 等 VC seats

### Coup Trigger
- Federal investigation on unlicensed broker activity
- Buzzfeed report 1/2016: "Zenefits sold insurance with unlicensed brokers"
- Compliance scandal + Parker 承认 "we built a tool to let brokers take the licensing test faster"

### Coup Move
- 2/2016 David Sacks becomes interim CEO
- Parker forced to resign
- Sacks email: "The problem is cultural"
- Board 主导 + Parker 个人 compensation negotiation

### CEO Response
- Initially try to stay
- Accept exit package
- Non-disparagement agreement
- Year later found Rippling (competitor)

### Outcome
- Parker out
- Sacks later departed
- Zenefits valuation crash to $2B
- Parker's next company Rippling 成功（valued $11B by 2024）

### Lessons
- **Regulatory scandals = CEO termination signal**
- **Parker's second act shows founder resilience**
- **Non-disparagement prevents you telling your side**

### 对 founder 的教训
- **Regulatory compliance is existential**——don't shortcut
- **Accept exit package, preserve runway for next venture**
- **Your brand can recover——Parker at Rippling proved it**

## 失败案例 5：Adam Neumann × WeWork（2019）

### Context
- WeWork 2019 S-1 IPO file
- Valuation $47B at peak, 降到 $8B
- Adam 的 governance issues 全部 exposed

### Board Structure
- SoftBank $18B invested，dominant position
- Multiple other VC + Adam 的 super-voting stock（20x common）

### Coup Trigger
- S-1 disclosure:
  - Self-dealing (Adam owned buildings leased to WeWork)
  - Loans from WeWork to Adam
  - Adam 的 wife 的 veto on successor CEO
  - Use of trademark "We" personally owned by Adam
- Media outcry
- IPO investors refuse to buy
- IPO pulled

### Coup Move
- 9/2019 Board 和 SoftBank 协议 remove Adam
- Adam 要求 exit package
- 10/2019 Adam step down as CEO
- Massive severance: $1.7B (shares + cash + consulting)

### CEO Response
- Negotiate exit package intensively
- Retain significant equity
- Non-compete relaxed
- Essentially "buy out" Adam's position

### Outcome
- WeWork IPO delayed 2 年
- Adam transition to Flow (new real estate startup)
- Flow raises $350M at $1B in 2022

### Lessons
- **Even toxic founder can negotiate large exit**——when leverage exists
- **Governance issues = IPO killer**
- **Founders with super-voting + disputes = time bomb**

### 对 founder 的教训
- **Governance matters at scale**——self-dealing 在 Seed 阶段可能 OK，IPO 阶段 = death
- **Super-voting 让你 entrenched 但也让 exit 更复杂**
- **Exit package negotiation 是 critical skill**

## 失败案例 6：Jack Dorsey × Twitter（2008）

### Context
- Twitter 早期，Jack 创始人 CEO
- 2006 公司成立，2008 Jack 被 fire

### Board Structure
- Evan Williams + Biz Stone + Jack
- Evan 最大 shareholder

### Coup Trigger
- Jack 的 operational effectiveness 被 questioned
- Evan 觉得 Jack 不是 operator
- Product 方向分歧

### Coup Move
- 10/2008 Evan Williams 替代 Jack 作 CEO
- Jack 被 push to Chairman
- Jack 的 operational role 消失

### CEO Response
- Accept role change
- Remain chairman + advisor
- Work on other projects (Square 2009)

### Outcome
- Jack out as CEO 2008
- Square 2009 成功
- 2015 Jack 回 Twitter 作 CEO (in crisis)
- 2021 Jack resign Twitter
- 2022 Elon Musk acquired

### Lessons
- **Product founder vs operator founder**——Jack 是前者
- **Return after exile 是 pattern**——但 needs company's crisis
- **Founder with multiple ventures**——Square 让 Jack 有 own path

### 对 founder 的教训
- **Know yourself: are you product or operator?**
- **Product founders 经常 fire 然后 recall**
- **Don't put all eggs in one basket**——Jack 有 Square

## 失败案例 7：Bob Swan × Intel（2021）

### Context
- Intel 老牌半导体 giant
- 面临 TSMC + Samsung 工艺落后
- Bob Swan CFO → interim CEO → CEO

### Board Structure
- Independent board of public company
- Long-tenured directors

### Coup Trigger
- Process manufacturing delays (10nm, 7nm)
- Loss of Apple as customer (M-series silicon)
- Activist investor Third Point pressure
- Missed key product roadmaps

### Coup Move
- 1/2021 Bob Swan replaced by Pat Gelsinger
- Announced "new technical direction"
- Third Point 的 letter 推动

### CEO Response
- Graceful exit
- Transition handled professionally

### Outcome
- Pat Gelsinger take over
- Intel continue 面对 challenges
- 2024 Pat himself 面对 pressure

### Lessons
- **Public company CEO termination is different**——more formal but similar pressure
- **Technical miss = career ending**——especially in tech
- **Activist investors = modern boardroom coup**

## 成功案例 1：Mark Zuckerberg × Facebook（defending 2010s）

### Context
- Facebook 2010+ 面对多种 pressure
- Super-voting stock 让 Mark 维持 control
- 多次 board push for changes

### Board Structure
- Class B super-voting shares 10:1 favored Mark
- Mark 永远有 majority voting

### Defense Mechanism
- **Dual-class stock** 是 ultimate defense
- Independent directors 的 challenge 都 blocked
- IPO 前 set 好 structure

### Outcome
- Mark continue CEO 直到今天（2024+）

### Lessons
- **Dual-class structure pre-IPO = permanent defense**
- **Public company 也能 block coup if structure right**
- **Gates-Ballmer-Allen 和 Zuckerberg 都 use same pattern**

### 对 founder 的教训
- **If possible, 在 TS 时 fight for dual-class**
- 这是 controversial 但 effective

## Cross-Case Patterns

### Pattern 1：Dual-class 是 ultimate defense
- Facebook, Google, Snap, Palantir 等 employ
- Controversial but effective
- 必须在 IPO 前 set 好

### Pattern 2：Cofounder alignment 倍增 defense
- Collison brothers (Stripe)
- Hoffman + Weiner (LinkedIn)
- Larry + Sergey (Google)
- Single founder 更 vulnerable

### Pattern 3：Scandal accumulation = death spiral
- Travis (Uber): 多个 scandal
- Neumann (WeWork): 多个 governance issue
- Conrad (Zenefits): regulatory + culture
- 1 scandal 可 survive，3+ scandal 不行

### Pattern 4：Employees as trump card
- Sam Altman (OpenAI): 500+ employees letter
- Steve Jobs (Apple): eventually return via Pixar → NeXT → Apple
- 需要 real loyalty, not performative

### Pattern 5：Return from exile requires company's failure
- Jobs 1997: Apple 接近 bankruptcy
- Dorsey 2015: Twitter in crisis
- 如果 公司 flourishing without you, you don't return

### Pattern 6：Product vs Operator founder
- Product founders (Jobs, Dorsey): 更容易被 replace 但经常 return
- Operator founders (Gates, Bezos): 更能 defend
- Identity matters for defense strategy

### Pattern 7：Exit package negotiation 是 critical skill
- $1.7B (Neumann), $90M (Rubin), $X (many others)
- Leverage = timing + employee loyalty + legal position

## Decision Tree for Founders

```
Coup signals detected?
├─ Have dual-class stock majority?
│  ├─ Yes: Just wait them out, you control votes
│  └─ No: Continue
├─ Have 2+ board members aligned with you?
│  ├─ Yes: Fight. Rally them + prepare counter-moves
│  └─ No: Continue
├─ Have operational leverage (key customers, key employees)?
│  ├─ Yes: Threaten nuclear option (like Sam Altman)
│  └─ No: Continue
├─ Company in crisis requiring you?
│  ├─ Yes: "We need the captain" argument
│  └─ No: Continue
├─ Scandal or performance issue?
│  ├─ Yes: Negotiate exit package (preserve equity + severance)
│  └─ No: Try "cool down" mediation + accept some concessions
```
